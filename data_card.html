Analyzing 83,000 Steam Games: Dataset

A look at the dataset behind the analysis.

Source

This dataset was sourced from Kaggle and contains information on 83,646 games listed on Steam as of May 2024. It was compiled by scraping the Steam store and SteamSpy APIs.

View the dataset on Kaggle

What the data contains

Each row represents a single game. Key columns used in this analysis include release date, Windows/Mac/Linux support, DLC count, and average playtime. The full dataset also includes fields like price, genres, tags, estimated owners, and review scores.

Cleaning

Release dates were converted to datetime format, and rows with missing dates were filtered out. The analysis was then filtered to only include the 2010–2023 year range.
